# awesome-tamil - சிறப்பான சிக்கலான தமிழ்
தமிழில் உள்ள **பொதுவெளி தரவுகள்**, நிரல் திரட்டுகள், மற்றும் மென்பொருள்கள். கற்றது கைமண்ணளவு - எதுவும் விடுபட்டிருந்தால் நிவர்த்திசெய்யவும்.
மற்றதரவுகள் இருப்பினும், இவை தனிமனிதர் கணினியாளர்களுக்கு எட்டாக் கனியாகவே உள்ளது - காலும் மாறக்கூடும் - அப்போது யாரேனும் இந்த பட்டியலை புதுப்பித்து கிட்-இல் புதுப்பதிவு அனுப்புங்கள்.
